# 📊 Análise de Vendas com Python

Projeto de análise de vendas utilizando Python, com geração de relatórios e dashboard gráfico.

## 🚀 Funcionalidades

- 📊 Dashboard com Matplotlib
- 💰 Cálculo de faturamento total
- 🧾 Ticket médio
- 🏆 Top produtos
- 📦 Análise por categoria
- 📅 Faturamento por dia

## 🛠️ Tecnologias

- Python
- Matplotlib
- Collections (Counter, defaultdict)

## 📈 Objetivo

Simular um sistema de Business Intelligence (BI) para análise de vendas de uma loja de roupas masculinas.

Projeto desenvolvido para estudo de Data Analysis com Python(Faculdade).
